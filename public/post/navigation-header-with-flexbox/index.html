Navigation Header with Flexbox

2/Oct 2015 By Ryan Jordan

I decided to continue with the small flexbox tutorials. So here is another one. In this tutorial, we will make a navigation header using flexbox. If you haven’t read my first post on flexbox, you should check it out here. I have made a codepen of what we will be building. Let’s get started with the HTML.

<body>
  <div id="header">
    <a href="#">
      <i class="fa fa-tree"></i>
    </a>
    <nav role='navigation'>
      <ul>
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
        <li><a href="#">Clients</a></li>
        <li><a href="#">Contact Us</a></li>
      </ul>
    </nav>
    <a href="#">Learn More
      <i class="fa fa-bars"></i>
    </a>
  </div>
</body>

Nothing new here, it is a very simple navbar using some icons from Font Awesome. I’ll style it up a bit so we can get to the point. I’m using SASS, so if you are still using regular CSS you should switch! If you are just starting out, check out the codepen and click the view compiled button to view the regular CSS. This will also show you all of the prefixers too (-webkit-, -moz, etc.).

#header {
  height: 2.5em;
  color: #333;
  a {
    text-decoration: none;
    color: #333;
    &:hover{
      color: lighten(#333, 30%);
    }
  }
  i {
    padding: 0 0.625em;
    font-size: 1.250em;
    &:hover {
      color: lighten(#333, 30%);
    }
  }
  nav {
    ul {
      li {
        list-style-type: none;
        padding: 0.625em 1.250em;
        border-right: 1px solid #e5e5e5;
        &:last-of-type {
          border: none;
        }
      }
    }
  }
}

So this will give us:

Adding Flexbox

So lets add some flexbox. We’ll start off by adding display: flex;, align-items: center; and justify-content: center; to the div#header, like so:

#header {
  display: flex;
  align-items: center;
  justify-content: center;
  ...
}

This centers the content inside the header. Note: the <ul> is pushing the tree icon to the left. No need to concern ourselves with that. Now lets style the <ul> to display: flex;, align-items: center; and justify-content. For the sake of this tutorial, and because I hate incomplete tutorials that skip over certain sections and assume the reader knows what they don’t know…

...
nav {
  ul {
    display: flex;
    align-items: center;
    justify-content: center;
    li {
      ...
    }
  }
}

Alright, so we are almost there! We now have our navbar and unordered list centered. Now we just need to figure out a way to push the tree icon to the left and the “Learn More” and menu icon to the right. We could do something crazy like mess with the width of the <nav> or use padding or margin. I dare say we could even use position: absolute;, but NO!! There is a better solution. Let’s look at flex-grow: 1;.

flex-grow is defined by CSS-Tricks as

The ability for a flex item to grow if necessary. It accepts a unitless value that serves as a proportion. It dictates what amount of the available space inside the flex container item should take up.

I personally use the flex shorthand property since it only requires one parameter, flex-grow. It will accomplish the same goal. So we will add flex: 1; to our <nav> element. Like so:

...

nav {
  flex: 1;
  ul {
    display: flex;
    align-items: center;
    ...

Our header will now be complete.

So without using excessive padding, margin, absolute positioning or a framework we have built a simple navigation header. It isn’t mobile friendly, but works well on medium and up views. We would need to do a bit more to make it responsive.

That’s all for now. The complete SASS file is written below or, once again, you can check out the codepen for the demo and code. #header {
  display: flex;
  align-items: center;
  justify-content: center;
  height: 2.5em;
  color: #333;
  a {
    text-decoration: none;
    color: #333;
    &:hover {
      color: lighten(#333, 30%);
    }
  }
  i {
    padding: 0 0.625em;
    font-size: 1.250em;
    &:hover {
      color: lighten(#333, 30%);
    }
  }
  nav {
    flex: 1;
    ul {
      display: flex;
      align-items: center;
      justify-content: center;
      li {
        list-style-type: none;
        padding: 0.625em 1.250em;
        border-right: 1px solid #e5e5e5;
        &:last-of-type {
          border: none;
        }
      }
    }
  }
}
